## 🧩 什么是依恋类型？

依恋风格是指我们在亲密关系中的行为和情感模式。ECR量表主要评估两个维度：

| 维度 | 含义 |
|------|------|
| **焦虑** | 对被抛弃的担忧程度 |
| **回避** | 与他人保持距离的倾向 |

根据这两个维度的高低组合，分为四种依恋类型：

| 类型 | 特点 |
|------|------|
| **安全型** | 享受亲密关系，信任伴侣 |
| **焦虑型** | 渴望亲密，但担心被抛弃 |
| **回避型** | 独立自主，不太依赖他人 |
| **恐惧型** | 渴望亲密但又害怕受伤 |

## 📚 关于ECR量表

本小程序使用的ECR（Experiences in Close Relationships）量表是由心理学家Fraley、Waller和Brennan于2000年开发的经典心理测量工具，在心理学研究中被广泛使用。
